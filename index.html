Linguagem de Programação

Uma linguagem de programação é um conjunto de símbolos e códigos usados para orientar a programação de estruturas no desenvolvimento da web.

CPU

CPU é a sigla para Central Process Unit, ou Unidade Central de Processamento. Ele é o principal item de hardware do computador, que também é conhecido como processador. A CPU é responsável por calcular e realizar tarefas determinadas pelo usuário e é considerado o cérebro do PC.

Memória

Em informática, memória são todos os dispositivos que permitem a um computador guardar dados, temporária ou permanentemente. Memória é um termo genérico para designar componentes de um sistema capazes de armazenar dados e programas. O conceito de computador digital binário com programa armazenado (arquitetura de Von Neumann e subsequentes) é sempre baseado no uso de memória, e não existiria sem a utilização destas.

Internet

A Internet é um sistema global de redes de computadores interligadas que utilizam um conjunto próprio de protocolos (Internet Protocol Suite ou TCP/IP) com o propósito de servir progressivamente usuários no mundo inteiro.
